# Smart City Complaint Management System

## User Manual

---

## 1. Introduction

This guide explains how to use the Smart City Complaint Management System to submit, track, and manage urban complaints.

---

## 2. User Roles

- Citizen: Submits and tracks complaints
- Agent: Processes complaints
- Admin: Manages users and system configuration

---

## 3. System Access

1. Open the web application in a browser.
2. Create an account or log in.
3. Access the dashboard according to your role.

---

## 4. Citizen Features

### 4.1 Create an Account

- Click "Register"
- Enter name, email, and password
- Submit the form

### 4.2 Submit a Complaint

- Click "New Complaint"
- Fill category, description, urgency, and location
- Upload photos (optional)
- Submit

### 4.3 Track Complaint Status

- Open "My Complaints"
- View current status and history

---

## 5. Agent Features

### 5.1 View Assigned Complaints

- Access dashboard
- Filter by status or category

### 5.2 Update Complaint Status

- Open complaint
- Change status to In Progress or Resolved

### 5.3 Add Internal Notes

- Add processing notes visible only to staff

---

## 6. Admin Features

### 6.1 Manage Users

- Create, update, or deactivate accounts
- Assign roles

### 6.2 Manage Departments

- Add or edit service departments

---

## 7. Notifications

- Users receive notifications when complaint status changes
- Delivered via in-app alerts and email (if enabled)

---

## 8. Error Messages & Troubleshooting

| Issue            | Solution                   |
| ---------------- | -------------------------- |
| Login failed     | Verify credentials         |
| Upload failed    | Check file size and format |
| Page not loading | Refresh browser            |

---

## 9. Security & Privacy

- Personal data is encrypted
- Access controlled by role
- Only authorized users can view complaints

---

## 10. Support

For assistance, contact the system administrator or consult project documentation.
